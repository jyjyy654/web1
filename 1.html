Hypertext Markup Language (HTML) is the standard Markup Language for creating web pages and web applications. This tutorial shows you
how to add a line break in HTML. By default, browsers ignore many formatting
keystrokes that we take for granted. Examples include the “Enter” and “Tab” keys and
multiple uses of the spacebar. To accomplish the same tasks in HTML, you use page formatting tags.
